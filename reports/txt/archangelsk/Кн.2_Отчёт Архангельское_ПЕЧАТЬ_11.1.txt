11.2. Обоснование подсчетных параметров 
11.2.1. Определение площадей и объемов нефтенасыщенных пород 
Определение площадей нефтеносности и объемов нефтенасыщенных пород-коллек-
торов проводилось с помощью пакета программ IRAP RMS (Roxar) с выводом графических 
приложений, в том числе структурных карт по кровле коллектора и карт эффективных 
нефтенасыщенных толщин в масштабе 1:25000, на которые нанесены границы залежей и 
категорий подсчета (Граф. 5-30).  
Площадь нефтенасыщенных пород залежей определялась на подсчетных планах, 
совмещенных со структурными картами по кровле коллекторов и ограниченных внешним 
контуром нефтеносности, также учитывался внутренний контур нефтеносности, который 
выделялся на структурных картах по подошве коллектора.  
Контур нефтеносности залежей нефти шешминского горизонта определялся по ре-
зультатам построения схематических геологических профилей. Точки пересечения абсо-
лютных отметок подошвы залежи с ее кровлей принимались за нулевые нефтенасыщенной 
толщины. Совокупность всех полученных точек и служила определением контура залежей. 
Площадь по залежам рассчитывалась раздельно для каждого продуктивного пласта 
по поднятиям, залежам, категориям, лицензиям, зонам. Внешний контур нефтеносности за-
лежей принимался за нулевую изопахиту. 
Объемы залежей определялись по картам эффективных нефтенасыщенных толщин 
раздельно для каждого продуктивного пласта по поднятиям, залежам, категориям, зонам, в 
соответствии с классификацией 2013 г. по категориям А, В1 и В2.

Вычисление нефтенасыщенных толщин проводилось по комплексу промыслово-
геофизических исследований с учетом результатов опробования и данных керна. Методика 
выделения нефтенасыщенных толщин в терригенных и карбонатных коллекторах подробно 
изложена в пятом разделе данного отчета. На основании выделенных в разрезах скважин 
эффективных толщин строились карты этого параметра для интервалов нефтеносных пла-
стов путем интерполяции и экстраполяции скважинных данных. Затем, с учетом положения 
внутренних и внешних контуров, составлялись карты эффективных нефтенасыщенных тол-
щин, по которым определялись площади и объемы нефтенасыщенных пород. Средневзве-
шенное значение нефтенасыщенной толщины для всех залежей месторождения рассчиты-
вались путем деления суммарного объема нефтенасыщенных пород залежей на ее площадь 
в целом.  
Площадь и, соответственно, объем нефтенасыщенных пород залежей нефти шеш-
минского горизонта рассчитывались в целом по залежи с толщиной 3 м и более. Принятые 
при подсчете запасов эффективные нефтенасыщенные толщины песчаной пачки шешмин-
ского горизонта уфимского яруса приведены в приложении 10. 
11.2.2. Коэффициент пористости 
Коэффициент пористости (Кп) для нефтенасыщенных продуктивных отложений 
определялся как по образцам керна в лабораторных условиях, так и по промыслово-геофи-
зическим данным (нейтронный гамма-метод – методика двух опорных горизонтов).  
Пористость коллекторов по керну определялась в целом по продуктивным отложе-
ниям (текстовые приложения 25-29), по шешминскому горизонту – раздельно по залежам, 
по данным ГИС – как в целом по горизонту (ярусу), так и раздельно по залежам (текстовые 
приложения 30-38). Обоснование принятых в настоящем подсчете значений пористости, их 
сопоставление с ранее утвержденными (ПЗ 2003 г.) и числящимися на госбалансе приве-
дены в таблице 11.1. 
В пределах шешминского горизонта установлено четыре залежи нефти на четырех 
поднятиях. Керн был поднят из скважин на двух поднятиях. Коэффициент пористости опре-
делялся по материалам керна и данным ГИС. Среднеарифметическое значение открытой 
пористости по керну для залежи СВН Архангельского поднятия составило 29.7 % (1003 
определения в 65 скважинах), для залежи нефти Киярлинского поднятия – 29,4 % (14 опре-
делений в трех скважинах). Средневзвешенное значения пористости коллекторов для за-
лежи Архангельского поднятия по ГИС равно 29.5 % (215 определений в 61 скважине).
