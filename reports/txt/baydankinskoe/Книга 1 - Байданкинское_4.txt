170 4 ЦИФРОВЫЕ МОДЕЛИ МЕСТОРОЖДЕНИЯ 4.1 Цифровая геологическая модель месторождения Геологическое моделирование продуктивных отложений верейского горизонта, башкирского яруса, тульско-бобриковских отложений и турнейского яруса Байданкинского месторождения осуществлялось с помощью программного комплекса RMS компании Roxar. 4.1.1. Сбор и подготовка исходных данных Создание базы геолого-геофизических данных Для построения геологических моделей залежей Байданкинского месторождения собраны и подготовлены следующие исходные данные: − скважинные данные: номер скважины, альтитуда ротора, координаты устья, глубина забоя; − данные замеров кривизны по скважинам; − ГИС, результаты интерпретации данных ГИС: параметры, определенные в прослоях коллекторов, такие как индекс литологии (1 – коллектор, 0 – неколлектор), пористость, нефтенасыщенность; − глубина кровли и подошвы продуктивных пластов; − данные об абсолютных отметках водонефтяных контактов; − структурные карты по кровлям продуктивных отложений; − карты эффективных нефтенасыщенных толщин по продуктивным пластам. 4.1.2. Для согласования между собой поверхностей кровель и подошв пластов построение поверхностей (начиная с базовой поверхности) велось методом схождения. Осреднение и перенос скважинных данных на сеточную область Следующим этапом геологического моделирования является построение скважинной модели, т.е. осуществляется процесс перевода скважинных данных в ячейки трехмерной сетки. Осреднение данных происходило по интервалам между соседними замеренными точками. Границы интервалов – середина расстояния между точками с данными. Для каждого типа скважинных данных выбирался соответствующий метод осреднения. При осреднении дискретной кривой, содержащей код литологии (коллектор– неколлектор), для заполнения каждой ячейки использовалась весовая функция выбора преобладающего значения кода. 4.1.5. Построение литологических моделей залежей и распределения фильтрационно-емкостных свойств пластов Параметрическое моделирование включает две стадии: сначала моделируется пространственное распределение фаций (строится литологическая модель), а затем распределение значений петрофизических параметров. 177 Построение литологических моделей продуктивных отложений проходило в два этапа. На первом этапе распределялся непрерывный параметр коллектора LITO_C, рассчитанный вдоль ствола скважин по результатам интерпретации ГИС, представляющий собой преобразованный в непрерывный вид дискретный параметр LITO_D (код «коллектор –неколлектор»). На втором этапе каждой ячейке модели присваивался код коллектора или неколлектора. Критерием отнесения ячеек к той или иной группе пород являлось граничное значение, равное 0,5. Если величина параметра коллектора в ячейке составляла меньше граничного значения, ячейка относилась к неколлектору, если больше граничного значения – к коллектору. В результате получен трехмерный параметр, характеризующий долю коллекторов, а также трехмерное распределение кода «коллектор – неколлектор», где каждой ячейке присваивался код 1 или 0. Для продуктивных отложений поле распределения коллекторов было получено с помощью метода «Prediction» (панель «Petrophysical modelling»). Контроль построения модели литологии проводился путем сопоставления распределения дискретного параметра «коллектор-неколлектор» с усредненным параметром по стволу скважины, сравнением геолого-статистических разрезов (рис. 4.1.164.1.26). На начальном этапе смоделирован 3D тренд нефтенасыщенности, рассчитанный через параметр водонасыщенности. Для моделирования использовались следующие параметры: пористость, проницаемость, высота над ВНК, взятые из подсчёта запасов. При распределении учитывались следующими трансформации: − минимальное и максимальное значения параметра, которые определялись по скважинным данным, − среднее значение параметра, которое определялось по скважинным данным − 3D тренд. Подсчет геологических запасов УВС По результатам трехмерного геологического моделирования проведен расчет геологических запасов нефти. В ячейках цифровых моделей, расположенных выше ВНК, рассчитывались значения объемов углеводородов. Затем объемным методом вычислялись соответствующие залежам величины запасов нефти (Q): Q=V·Кп·Кн·ρ·θ, где V – объем коллекторов, тыс. м3; Кп – коэффициент пористости, доли ед.; Кн – коэффициент нефтенасыщенности, доли ед.; ρ – плотность нефти, г/cм3; θ – пересчетный коэффициент, доли ед.; Геометрический объем коллекторов (V), подсчетные параметры (Кп, Кн) рассчитывались и перемножались как отдельные ячейки трехмерной сетки распределения параметров.
